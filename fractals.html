Koch Curve

Standard Koch Curve:

Modified Koch Curve:

The differences are:

  • Created a random angle between 50 and 70 degrees
  • Created a random number such that 1 in 5 lines disappear
        
def koch_curve(t, depth, length):
    t.pd()
    if depth == 1:
        rand = random.randrange(5)
        if rand != 1:
            t.fd(length)
        else:
            t.pu()
            t.fd(length)
            t.pd()
    else:
        low = random.randrange(50, 70)
        high = random.randrange(110,130)
        koch_curve(t,depth - 1,length)
        t.lt(low)
        koch_curve(t,depth - 1,length)
        t.rt(high)
        koch_curve(t,depth - 1,length)
        t.lt(low)
        koch_curve(t,depth - 1,length)

Sierpinski Triangle

Standard Sierpinski Triangle:

Modified Sierpinski Triangle:

The differences are:

  • The triangle alternates between blue and red after each recursion case. It starts and ends off with blue.
  • Alternates between a width of 4 and 1. It begins with a width of 4 and ends wit ha width of 1
  • Inserted another recursion step after the first 60 degree left turn
        
def triangle(t, size):
    t.lt(60)
    t.fd(size)
    t.rt(120)
    t.fd(size)
    t.rt(120)
    t.fd(size)
    t.rt(180)

def sierpinski(t, depth, size):
    t.speed(10)
    if depth == 1:
        triangle(t, size)
    else:
        t.clior("blue")
        sierpinski(t, depth-1, size/2)
        t.width(4)
        t.fd(size/2)
        t.width(1)
        sierpinski(t, depth-1, size/2)
        t.bk(size/2)
        t.lt(60)
        t.width(4)
        t.clior("red")
        sierpinski(t, depth-1, size/2)
        t.clior("blue")
        t.fd(size/2)
        t.width(1)
        t.rt(60)
        sierpinski(t, depth-1, size/2)
        t.rt(120)
        t.fd(size/2)
        t.lt(120)
